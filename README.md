# Online Learning Platform & Course Analytics (MySQL)

This project analyzes an online learning platform using MySQL.
It manages students, instructors, courses, and enrollments to
generate insights about course popularity and student engagement.

The system helps educational platforms understand enrollment trends,
instructor performance, and course demand through SQL analytics.

## Technologies Used
- MySQL 8.0
- SQL Analytics

## SQL Concepts Covered
- Joins
- Aggregate Functions
- Group By
- Subqueries
- Date-Based Analysis

## Key Features
- Student enrollment tracking
- Course-wise enrollment analytics
- Instructor performance insights
- Education data reporting

## Use Case
Suitable for e-learning platforms, EdTech companies, and academic
analytics systems.
